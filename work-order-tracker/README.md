# 📊 ISP Performance Tracker & Analytics Dashboard

[![Next.js](https://img.shields.io/badge/Next.js-14-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![Supabase](https://img.shields.io/badge/Supabase-Database-emerald?style=for-the-badge&logo=supabase)](https://supabase.com/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-CSS-blue?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)

Dashboard monitoring manajemen data ISP (Internet Service Provider) yang dirancang untuk melacak aktivasi, pemutusan, dan performa tim secara real-time. Dilengkapi dengan visualisasi data interaktif dan fitur export laporan profesional.



## 🚀 Fitur Utama

- **Real-time Analytics**: Visualisasi tren bulanan menggunakan ApexCharts (Area & Bar Charts).
- **Monthly Summary Report**: Ringkasan otomatis status pelanggan (Berlangganan, Cuti, Putus).
- **Team Performance Tracking**: Monitoring produktivitas tim teknisi secara detail.
- **Professional Excel Export**: Generate laporan .xlsx yang terstruktur rapi dengan satu klik.
- **Role-Based Access Control (RBAC)**: Keamanan data berdasarkan peran user (Viewer/Aktivator/Admin).
- **Responsive Design**: Tampilan optimal di desktop maupun perangkat mobile.

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Styling**: Tailwind CSS & Lucide Icons
- **Charts**: ApexCharts
- **Data Export**: SheetJS (XLSX)
